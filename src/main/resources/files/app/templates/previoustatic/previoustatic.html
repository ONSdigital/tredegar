Previous version page

This is ONS prototype website build

This link would provide access to previous versions of data or bulletins published by ONS on an earlier date. This page will work slightly differently dependent on if you have come from a data page (a time series or dataset) or a statistical bulletin.

Time series and datasets

Part of the statistical process can be to revise data as new information becomes available and these pages are designed to provide access to the data that was previously published. This allows the latest figures to maintain a static URL and give consistent access to the latest estimates from the same place. The following screenshot shows how the page will be displayed.

This page will keep a record of all updates to the data and allow users to download one or all of these for comparison. Where available and relevant we will also look to publish reasons for the change in the data.

Statistical bulletins

For statistical bulletins this page will provide links to all previously published versions of the bulletin you are looking at. When you are on a previous bulletin the link will be replaced with a ‘Latest version’ link taking you to the most recent bulletin.
