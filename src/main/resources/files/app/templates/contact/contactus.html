This is ONS Alpha website build.

We have done this to demonstrate what the ONS website could look like in the future. Throughout the process we have, and will continue to, test with real users of the website. This website will close on the 31 December 2014.

Content on this website is not conclusive and may not be up-to-date or correct. Please refer to the current ONS website for the most recent statistics.
