Accessibility

This is an experimental prototype for a Office for National Statistics website. The website doesn’t have a separate accessibility statement, however we’ve tried to design this prototype to be as accessible and usable as possible for every user.

We are aware of some accessibility limitations of the site, in particular relating to colour contrast, incorrectly ordered headings and hidden content. The site also currently uses an AngularJS framework which is not necessarily compatible with some screen readers, however this will not be the approach taken for developing the site in the future.

Compatibility with tools

This prototype website should be compatible with recent versions of the following screen readers:

  • Jaws
  • NVDA
  • Thunder
  • VoiceOver

The site should also be compatible with:

  • basic operating system screen magnifiers
  • speech recognition software, eg Dragon Naturally Speaking
  • operating system speech packages
